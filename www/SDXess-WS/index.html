User Registration Function:

URL:http://inet99.ji8.net/SDXess-WS/user/registration/
Request Method: POST
Content Type: application/json
JSON format:
{
  "registration_data":  {
      "user":  "sdxess3@ji8.net",
      "password":     "68808c1cac56078870fbdcf33d9fddd2",
      "fname":  "SDXess",
      "lname":     "Admin",
      "country":  "CN",
      "address":     "Haidian District, Beijing"
  }
}
Data format:
  • registration_data
    • user: E-mail Format, Min length: 5, Max length: 100
    • password: MD5 String (Alphanumeric), Min length: 32, Max length: 32
    • fname: (User's First name) Alphabetic, Min length: 2, Max length: 35
    • lname: (User's Last name) Alphabetic, Min length: 2, Max length: 35
    • country: Alphabetic: ISO 3166-1 standard for actual list of alphabetic codes and countries; Min length: 2, Max length: 3
    • address: (Billing Address)Text validation, Min length: 10, Max length: 500
Messages:
  • Status: 201, status_message: User Registered Successfully, data: User data in json format.
